# LPA-RBX-Inspector

**LPA-RBX-Inspector** is a simple external tool that reads Roblox memory.
It is **read-only** and does not write or modify anything in memory.

This is a rewrite of the original **LPA-RBX-Inspector v1**, with beter logic better memory reading and improved Roblox logic

## other

* updates offsets after Roblox updates
* Improved logic compared to v1
